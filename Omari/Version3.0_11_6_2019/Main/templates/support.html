Support

Information on how to properly use the application, and remediate errors. If you have been redirected here after attempting a Hate Speech search, note the error displayed to the left and proceed to the respective error documentation below. If you encounter an unspecified error and require assistance, please contact us. EraseHateData@gmail.com

Disclaimer

We have built this service to allow users a helping hand in identifying hate speech. Please keep in mind that the classifications we provide are predictions, and are not explicitly definitive. Please use discrection, consider all facts and intentions, before taking any actions based on the results provided.

    Instructions for classifying hate speech

  • 1. Enter a search term/keyword or text in the text input field.

  • 2. Specify what type of content was entered in the text input field.
    • Topic: Select this option if you would like to classify hate speech in tweets related to the search term you entered.
    • User: Select this option to classify hate speech in a specific users timeline(tweets from an individual user). Enter the exact screen name, with or without the @ symbol. Screen name input is not case sensitive.
    • Text: Select this option if you have text you would like classified. Works best with complete sentences, though any input that is a collection of words, will be classified without an error. Remove formatting and minimize whitespace when possible(slight increase in classification speed).

  • 3. Select a classification method. If you are unclear on the differences, select LSTM.
    • LSTM: Long Short-Term Memory. Uses a recurrent neural network for classification.
      • Current development model. This model will receive updates.
    • LogRegess/LinSVC: Logisitc Regression and Linear Support Vector Classification.
      • Legacy Model. No future updates planned.
        • Based on the works and publication of Davidson,Warmsley, Macy, and Weber
        Automated Hate Speech Detection and the Problem of Offensive Language.ICWSM '17, 2017

  • 4. Submit the form to receive results!
    • Be kind to the service.Preferably, allow 10-20 seconds between searches/classification requests, for consistent performance.

Errors


  • Twitter Error - Type: ____ - Input:____

    If you attempted a Topic search, your input returned no results from Twitter. Wait 10-20sec before retrying.
    If you attempted a User timeline search, your input returned no User results from Twitter. Check the screen name entered. It must be exactly as it appears on Twitter.Not case sensitive, with or without the @ symbol. Wait 10-20sec before retrying.

  • Twitter Server Error - Twitter search locked: Duration 15min

    Twitter responded to your search with an error or exception. This error was not deemed to be a result of incorrect user inputs. This error may arise for a variety of reasons: overloaded server, ratelimit rejections, Twitter server maintenance. To prevent server abuse, users receiving this error message will be limited to using ONLY the "text" input option, for a duration of 15 minutes.

  • Model Serving Error

    This generally occurs when the model and/or model server is undergoing maintenance or updates. Sudden increases in site traffic may also throw this error. If you have received this error, we are aware of the issue and working on it! Allow 30min-1hr before attempting another search.
